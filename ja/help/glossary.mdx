## 一般的な用語

<AccordionGroup>
  <Accordion title="プリント対応（水密）">
    完全にシールされたモデル — 穴、隙間、欠落したサーフェスがない状態。水を入れても漏れない風船のようなものです。プリンターは内部と外部を区別する必要があるため、3D印刷にはこれが必要です。
  </Accordion>
  <Accordion title="ポリゴン（面 / 三角形）">
    3Dモデルを構成する小さな平面。ほとんどの3Dモデルは数千〜数百万の三角形で構築されています。ポリゴンが多い＝ディテールが多い、ただしファイルも大きくなります。
  </Accordion>
  <Accordion title="頂点（複数：vertices）">
    ポリゴンのエッジが交わる角のポイント。3つの頂点で1つの三角形が定義されます。ポリゴン数と頂点数はどちらもモデルの複雑さの指標です。
  </Accordion>
  <Accordion title="法線">
    各サーフェスが向いている方向。法線はプリンター（およびビューア）にどちら側が外側でどちら側が内側かを伝えます。反転した法線はサーフェスを裏返しに見せます。
  </Accordion>
  <Accordion title="ノンマニフォールド">
    適切なソリッドを形成しないジオメトリ。例：3つ以上の面で共有されたエッジ、厚さゼロのサーフェス、空間に浮遊するルーズなエッジ。ノンマニフォールドジオメトリは印刷の失敗を引き起こします。
  </Accordion>
  <Accordion title="壁の厚さ">
    モデルの壁の厚さ。薄い壁は印刷中や後処理中に壊れる可能性があります。プリンターによって最小壁厚の要件が異なります。
  </Accordion>
  <Accordion title="中空化">
    モデル内部に空洞を作り、指定した厚さのシェルだけを残すこと。特に大きな印刷で材料を節約しコストを削減します。
  </Accordion>
  <Accordion title="オーバーハング">
    帽子のつばのように、下からの支えなしに外側に突き出ているモデルの部分。一定の角度（通常45°）を超えるオーバーハングは、印刷中にサポート構造が必要になる場合があります。
  </Accordion>
  <Accordion title="クリアランス">
    動く必要があるパーツや嵌め合わせるパーツ間の隙間。十分なクリアランスがないと、印刷中にパーツが融合してしまう場合があります。
  </Accordion>
  <Accordion title="ラティス">
    モデルの内部を充填するために使用される繰り返しの幾何学的パターン。ラティスは強度を維持しながら材料を節約します。一般的なパターンにはキュービックグリッドとジャイロイドサーフェスがあります。
  </Accordion>
  <Accordion title="ジャイロイド">
    すべての方向に繰り返す、滑らかで連続的に曲がったサーフェスの一種。ジャイロイドは重量に対して強く、美しい有機的な内部構造を生み出します。
  </Accordion>
  <Accordion title="UVマッピング">
    2D画像（テクスチャ）を3Dサーフェスにマッピングするシステム。プレゼント包装のようなものです：UVマップはソフトウェアに平面画像を3D形状にどのように巻き付けるかを指示します。
  </Accordion>
  <Accordion title="テクスチャ">
    3Dモデルのサーフェスに適用されるカラーとディテールを与える2D画像。テクスチャにはカラー、サーフェスディテール（凹凸）、マテリアルプロパティ（光沢 vs マット）を含むことができます。
  </Accordion>
  <Accordion title="メッシュ">
    3Dモデルのサーフェスを形成するポリゴン（三角形）の集合。「メッシュ修復」とは、ポリゴンサーフェスを印刷に適した状態に修正することを意味します。
  </Accordion>
  <Accordion title="シェル">
    モデル内の分離した独立したジオメトリの部品。1つのモデルに複数のシェルがある場合があります — 例えば、キャラクターモデルにはボディ、髪、衣服の別々のシェルがある場合があります。
  </Accordion>
  <Accordion title="スライサー">
    3Dモデルを3Dプリンターの命令（Gコード）に変換するソフトウェア。スライサーはモデルを薄い水平層に「スライス」します。人気のスライサーにはCura、PrusaSlicer、ChiTuBoxがあります。
  </Accordion>
  <Accordion title="STL">
    Stereolithographyファイルフォーマット。3D印刷で最も一般的なフォーマット。ジオメトリのみを含み、カラー、テクスチャ、マテリアルは含みません。
  </Accordion>
  <Accordion title="OBJ">
    Wavefront OBJファイルフォーマット。ジオメトリ、テクスチャ、マテリアル、カラーをサポート。フルカラー3D印刷で広く使用されています。
  </Accordion>
  <Accordion title="3MF">
    3D Manufacturing Format。3D印刷のために特別に設計された最新フォーマット。カラー、マテリアル、1ファイル内の複数オブジェクトをサポートします。
  </Accordion>
</AccordionGroup>

## プリンタータイプ

<AccordionGroup>
  <Accordion title="FDM（熱溶解積層法）">
    最も一般的なタイプの3Dプリンター。プラスチックフィラメントを溶かして層ごとに堆積します。手頃な価格で、幅広い材料選択肢があります。一般的な最小壁厚：1.0〜1.5 mm。
  </Accordion>
  <Accordion title="SLA（光造形法）">
    UVレーザーで液体レジンを層ごとに硬化させます。非常に細かいディテールを生成します。一般的な最小壁厚：0.5〜1.0 mm。
  </Accordion>
  <Accordion title="SLS（選択的レーザー焼結法）">
    レーザーで粉末材料を融合します。サポート構造が不要です。機能パーツに適しています。一般的な最小壁厚：0.7〜1.0 mm。
  </Accordion>
  <Accordion title="MJF（マルチジェットフュージョン）">
    HPの産業用粉末ベースの印刷技術。高速、高精度、良好な表面仕上げ。一般的な最小壁厚：0.5〜1.0 mm。
  </Accordion>
  <Accordion title="IJP（インクジェットプリンティング）">
    インクジェット技術を使用したフルカラー3D印刷。MimakiとStratasysのプリンターで使用。フォトリアリスティックなカラーモデルを生成します。一般的な最小壁厚：0.7〜1.0 mm。
  </Accordion>
</AccordionGroup>
