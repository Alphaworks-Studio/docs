</Frame>

修復を開始する際の最初のステップは、モデルタイプのプリセットを選択することです。各プリセットは、特定の種類の3Dモデルに合わせて修復プロセスを最適化します。

<CardGroup cols={2}>
  <Card title="キャラクター" icon="user">
    フィギュア、キャラクター、有機的な形状に最適化されています。顔の特徴、髪、衣服のしわなどの細かいディテールを保持します。繊細なジオメトリを維持するために、より穏やかなスムージングを使用します。
  </Card>
  <Card title="メカニカル" icon="gear">
    エンジニアリングパーツ、ブラケット、機能的なプリントに最適です。シャープなエッジとフラットな面を優先します。組み合わせが必要なパーツの正確な寸法を維持します。
  </Card>
  <Card title="ビークル" icon="car">
    自動車、航空機、その他の乗り物モデル向けに調整されています。滑らかなボディカーブと、ホイールやパネルなどの構造的ディテールのバランスを取ります。
  </Card>
  <Card title="カスタム" icon="sliders">
    プリセットによる最適化はありません。すべての設定を完全にコントロールできます。他のカテゴリに当てはまらないモデルに適した出発点です。
  </Card>
</CardGroup>

<Tip>
  どれを選べばよいかわからない場合は、**カスタム**から始めましょう。ほとんどのモデルでうまく機能します。後からいつでも別のプリセットで再修復できます。
</Tip>

<Note>
  AIアシスタントがモデルを分析済みの場合、推奨されるモデルタイプが提案されることがあります。修復ダイアログに**推奨**バッジが表示されます。
</Note>
