エディター右側のツールバーは、18のツールを区切り線で分けた3つのグループに整理しています — 8つのパイプラインツール、6つのインタラクティブツール、4つの検査ツールです。最後の区切り線は、クイックアクションの下段を分けています：スライスビュー、詳細、修復設定（修復の調整中のみ表示）です。

## パイプラインツール（上部）

これらの設定は**修復プロセス**に影響します。修復を実行する前に調整してください。

| ツール | 機能 |
|------|-------------|
| [スケール](/editor/tools/scale) | モデルのサイズ変更 |
| [壁の厚さ](/editor/tools/thickness) | 最小壁厚の設定 |
| [中空化](/editor/tools/hollow) | 中空の内部を作成 |
| [穴埋め](/editor/tools/close-holes) | 隙間や穴を埋める |
| [透明部分のカット](/editor/tools/cut-transparent) | 不可視のサーフェスを除去 |
| [カラー](/editor/tools/colors) | テクスチャ処理の設定 |
| [スムーズ](/editor/tools/smooth) | 修復中のサーフェススムージング |
| [ターゲット面数](/editor/tools/target-faces) | ポリゴン数の制御 |

## インタラクティブツール（中部）

これらのツールは**モデルを直接変更**します。修復後に追加の変更を行うために使用します。

| ツール | 機能 |
|------|-------------|
| [カット](/editor/tools/cut) | オプションのキー付きでピースに分割 |
| [ラティス](/editor/tools/lattice) | インフィル構造を生成 |
| [ラベル](/editor/tools/label) | サーフェスにテキストを追加 |
| [ミラー](/editor/tools/mirror) | ミラーコピーを作成 |
| [測定](/editor/tools/measure) | 距離を測定 |
| [小パーツ除去](/editor/tools/remove-small-parts) | 浮遊するデブリを除去 |

## 検査ツール（下部）

これらのツールはモデルを変更せずに**分析・可視化**します。

| ツール | 機能 |
|------|-------------|
| [検証](/editor/tools/validate) | 印刷適性の分析 |
| [エラー](/editor/tools/errors) | メッシュ品質レポート |
| [オブジェクト情報](/editor/tools/object-info) | 寸法と統計 |
| [360°ビュー](/editor/tools/360-view) | ライティングと表示オプション |

## 下段

最後の区切り線で分けられており、これらはツールグループではなくクイックアクションです。

| アクション | 機能 |
|------|-------------|
| [スライスビュー](/editor/tools/slice-view) | 断面の可視化 |
| 詳細 | モデルの寸法と修復の詳細 |
| 修復設定 | 現在の修復に使用された設定を確認 — 修復の調整中のみ表示 |

## 視覚的インジケーター

- **青色ハイライト** — 現在アクティブなツール
- **青いドット** — デフォルトから設定がカスタマイズされたツール
- **淡色表示** — ラティスのみが淡色表示になり、それは読み込まれたモデルが別のツールの出力（カット、ミラー、またはラベルの結果）である場合のみです。ラティスは元のモデルまたは修復済みモデルに適用できます。他のすべてのツールは、修復済みかどうかにかかわらず、あらゆるモデルで機能します。
