オブジェクト情報は、モデルの主要な測定値と統計をひと目で表示します。

## 表示される情報

| 指標 | 説明 |
|--------|------------|
| **寸法** | 選択した単位での幅（X）、高さ（Y）、奥行き（Z） |
| **ポリゴン数** | 三角形面の総数 |
| **頂点数** | 頂点の総数 |
| **体積** | cm³で計算された体積 |
| **表面積** | 外部表面積の合計 |
| **シェル数** | 分離した独立パーツの数 |

## 単位の切り替え

**ミリメートル（mm）**、**センチメートル（cm）**、**インチ**を切り替えて、お好みの単位で寸法を表示できます。

## ヒント

- **体積**は材料コストの見積もりに便利 — 多くの印刷サービスは体積で課金します
- **シェル数が多い**場合は浮遊パーツの可能性があります — [小パーツ除去](/editor/tools/remove-small-parts)でクリーンアップしてください
- 修復前後のポリゴン数を比較して、ジオメトリがどの程度最適化されたか確認してください
- 特定の特徴間の距離については、代わりに[測定](/editor/tools/measure)ツールを使用してください
