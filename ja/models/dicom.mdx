**DICOM**は医療画像の標準フォーマットで、病院の機器からエクスポートされるCTおよびMRIスキャンです。3D Prep Proは`.dcm`および`.dicom`ファイルを受け付け、スキャンを組織タイプごとに分離することで印刷可能な3Dモデルに変換します。

通常のメッシュとは異なり、DICOMスキャンは密度測定値のスタックであり、サーフェスではありません。修復する前に、**どの密度をソリッドパーツに変換するか**を3D Prep Proに指定します。これをセグメンテーションと呼びます。

## スキャンをセグメント化する

DICOMファイルをアップロードすると、**DICOM Segmentation**ダイアログが自動的に開きます。このステップは**必須**です。セグメンテーションのないスキャンは修復できません。

密度は**ハウンスフィールド単位（HU）**で測定されます。定義した各セグメントは、そのHU範囲内のすべてから構築された個別の水密パーツになります。

<Info>
  分析を実行する前にセグメンテーションを完了する必要があります。セグメント化されていないDICOMモデルを修復しようとすると、まずセグメンテーションプリセットで再分析するよう求められます。
</Info>

### プリセット

プリセットから始めて、必要に応じて範囲を調整します。

| プリセット | 作成されるセグメント |
| --- | --- |
| **Standard** | 硬組織（200–3000）、高密度軟組織（80–200）、軟組織（20–80）、脂肪（−120–20）、外側組織（−500〜−120） |
| **Head** | 頭蓋骨（200–3000）、脳（20–45）、皮膚（−200–20） |
| **Simple** | 硬組織（200–3000）、中密度組織（15–200）、外側組織（−200–15） |
| **Dense Only** | 高密度（700–3000）、低密度（200–700） |

### セグメントを定義する

<Steps>
  <Step title="プリセットを選択">
    印刷したいものに最も近いプリセット（骨、頭部全体など）を選択します。
  </Step>
  <Step title="範囲を調整">
    各セグメントについて、**密度 最小**と**密度 最大**（HU）を設定します。最小は最大より小さくする必要があり、各セグメントには名前が必要です。
  </Step>
  <Step title="セグメントの追加または削除">
    別の組織タイプを分離するには**セグメントを追加**を使用するか、不要なセグメントを削除します。
  </Step>
  <Step title="分析を開始">
    **分析を開始**をクリックします。各セグメントは独自の3Dパーツに再構成されます。
  </Step>
</Steps>

### ハウンスフィールド単位リファレンス

範囲設定に役立つ一般的な組織密度：

| 組織 | 範囲 |
| --- | --- |
| 皮質骨 | 700〜3000 HU |
| 海綿骨 | 200〜700 HU |
| 筋肉 | 10〜40 HU |
| 脳（灰白質 / 白質） | 20〜45 HU |
| 血液 | 30〜45 HU |
| 軟組織 | 20〜80 HU |
| 脂肪 | −120〜−50 HU |
| 肺 | −700〜−500 HU |
| 空気 | −1000 HU |
| 金属インプラント | 3000 HU以上 |

## DICOMモデルを修復する

DICOM修復は作成したセグメントに対して機能します：

- 修復ダイアログで**どのセグメントを修復するか**を選択します。選択した各セグメントは完全な修復パイプラインを独立して通過するため、修復時間はセグメント数とともに増加します。
- DICOMモデルは、通常のメッシュに使用される最大ディテール設定ではなく、スキャンデータ用に調整された**最適化された解像度**で修復されます。いずれにせよ、スキャン自体のスライス解像度が制限要因となります。
- 結果が階段状にならないよう、サーフェススムージングが自動的に適用されます。[再構築後スムージング](/repair/advanced-settings#smoothing)を0のままにすると、エンジンはDICOMモデルに対してこれを引き上げます。

<Tip>
  印刷する予定のセグメントのみを選択してください。5つのセグメントを修復するには、1つを修復する場合の約5倍の時間がかかります。
</Tip>

## ヒント

- 骨（硬組織）は最も印刷しやすく、密度の境界が最も明確です。
- パーツに穴や隙間が出る場合は、そのHU範囲を少し広げます。分離したかった組織が結合する場合は、範囲を狭めます。
- 修復後に[スライスビュー](/editor/tools/slice-view)を使用して、各セグメントの内部構造を確認します。
