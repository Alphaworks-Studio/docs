修复完成后，只需几次点击即可下载结果。

<Steps>
  <Step title="前往修复详情页面">
    修复完成后您会自动跳转到此页面。也可以从模型详情页面的"模型修复"选项卡中找到。
  </Step>
  <Step title="点击下载">
    点击标题区域的圆形 **下载** 按钮。这将一键下载包含所有格式（GLB、OBJ、STL 和 3MF）的打包文件——文件将在新标签页中打开。
  </Step>
  <Step title="只需要单一格式？打开 3D 编辑器">
    修复详情页面始终会将所有格式一起下载。要获取某个特定格式，请在 **3D 编辑器** 中打开结果并使用 **导出** 下拉菜单，它可以让您单独下载 **GLB**、**OBJ**、**STL** 或 **3MF**（或 **全部导出**）。切割、晶格、标签和镜像等工具操作也会在其操作页面上直接提供按格式选择的菜单。

    <Frame>
      <img src="/images/getting-started/download-menu.png" alt="3D 编辑器中的导出格式下拉菜单" />
    </Frame>
  </Step>
</Steps>

## 导出格式

| 格式 | 纹理 | 颜色 | 适用场景 |
|------|------|------|----------|
| **OBJ** | 是 | 是 | 全彩 3D 打印（Mimaki、Stratasys）。通用格式。 |
| **STL** | 否 | 否 | 标准 FDM/SLA 单色打印。切片软件兼容性最广。 |
| **GLB** | 是 | 是 | 分享、网页查看器、AR/VR。紧凑的单文件，内嵌纹理。 |
| **3MF** | 是 | 是 | 多材料和全彩打印机。现代推荐格式。 |

<Tip>
  对于 **Mimaki 全彩打印机**，使用 **3MF** 或 **OBJ** 以保留颜色和纹理。对于单色 FDM/SLA 打印机，**STL** 是标准选择。
</Tip>

<Note>
  如果需要，您也可以从模型详情页面下载 **原始**（未修复）模型。
</Note>
