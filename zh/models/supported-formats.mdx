## 导入格式

| 格式 | 扩展名 | 纹理 | 顶点颜色 | 多对象 | 备注 |
|------|--------|------|----------|--------|------|
| Wavefront OBJ | `.obj` + `.mtl` | 是 | 否 | 是 | 最常用。将 .mtl 和纹理文件放入 ZIP 中。 |
| 立体光刻 | `.stl` | 否 | 否 | 否 | 仅几何数据。3D 打印机广泛使用。 |
| 3D Manufacturing | `.3mf` | 是 | 是 | 是 | 现代格式，内嵌颜色和材质。 |
| Autodesk FBX | `.fbx` | 是 | 否 | 是 | 常用于游戏和动画工作流程。 |
| Stanford PLY | `.ply` | 否 | 是 | 否 | 点云和网格格式，支持顶点颜色。 |
| glTF / GLB | `.gltf`, `.glb` | 是 | 否 | 是 | Web 标准。GLB 将纹理捆绑在一个文件中。 |
| VRM (VRoid) | `.vrm` | 是 | 否 | 是 | 动漫/VTuber 角色格式。 |
| VRML | `.wrl` | 是 | 是 | 是 | 旧版 3D Web 格式。 |
| DICOM | `.dcm` | 否 | 是 | 否 | 医学影像（CT、MRI）。参见 [特殊配置](/models/uploading-files#医学影像dicom)。 |
| LAS / LAZ | `.las`, `.laz` | 否 | 是 | 否 | LiDAR 点云。参见 [特殊配置](/models/uploading-files#lidar-点云laslaz)。 |

## 导出格式

| 格式 | 纹理 | 颜色 | 适用场景 |
|------|------|------|----------|
| **OBJ** | 是 | 是 | 全彩 3D 打印（Mimaki、Stratasys）。通用格式。 |
| **STL** | 否 | 否 | 标准 FDM/SLA 单色打印。切片软件兼容性最广。 |
| **GLB** | 是 | 是 | 分享、网页查看器、AR/VR。紧凑的单文件。 |
| **3MF** | 是 | 是 | 多材料和全彩打印机。现代推荐格式。 |

## 压缩文件格式

上传 **ZIP**、**RAR** 或 **7Z** 文件，我们会自动解压。我们会找到其中的 3D 文件以及关联的纹理和材质。

<Tip>
  上传 **OBJ** 文件时，请将 `.mtl` 材质文件和所有纹理图片（`.png`、`.jpg`）放在同一个 ZIP 压缩包中。没有这些文件，模型上传后将没有颜色或纹理。
</Tip>

<Info>
  **DICOM** 和 **LAS/LAZ** 文件上传后会显示特殊配置对话框，您可以调整导入设置，如密度阈值和重建质量。详情请参阅 [上传文件](/models/uploading-files)。
</Info>

## 格式推荐

- **全彩打印**（Mimaki、Stratasys）：上传 OBJ+MTL 或 3MF。导出为 3MF 或 OBJ。
- **单色打印**（FDM、SLA）：上传任意格式。导出为 STL。
- **分享或网页查看**：导出为 GLB。
- **医学数据**：直接上传 DICOM 文件。
- **地形/测量数据**：上传 LAS 或 LAZ 文件。
