## 支持的文件格式

3D Prep Pro 支持所有主流 3D 文件格式：

| 格式 | 扩展名 | 纹理 | 颜色 |
|------|--------|------|------|
| Wavefront OBJ | `.obj` | 是 | 是 |
| 立体光刻 | `.stl` | 否 | 否 |
| 3D Manufacturing | `.3mf` | 是 | 是 |
| Autodesk FBX | `.fbx` | 是 | 是 |
| Stanford PLY | `.ply` | 否 | 是 |
| glTF / GLB | `.gltf`, `.glb` | 是 | 是 |
| VRM (VRoid) | `.vrm` | 是 | 是 |
| VRML | `.wrl` | 是 | 是 |
| DICOM（医学） | `.dcm` | 否 | 是 |
| LAS/LAZ (LiDAR) | `.las`, `.laz` | 否 | 是 |

您还可以上传 **ZIP**、**RAR** 和 **7Z** 压缩文件 — 我们会自动提取并处理其中的 3D 文件。

<Tip>
  需要帮助？在编辑器中打开 **AI 助手**，询问任何关于您模型的问题。它可以分析问题、建议解决方案，甚至为您执行工具。
</Tip>
