编辑器右侧的工具栏将19个工具分为三组，用分隔线隔开。

## 管线工具（上方）

这些设置影响**修复过程**。在运行修复之前进行调整。

| 工具 | 功能 |
|------|------|
| [缩放](/editor/tools/scale) | 调整模型大小 |
| [壁厚](/editor/tools/thickness) | 设置最小壁厚 |
| [挖空](/editor/tools/hollow) | 创建中空内部 |
| [关闭孔洞](/editor/tools/close-holes) | 填充缝隙和开口 |
| [裁剪透明](/editor/tools/cut-transparent) | 移除不可见表面 |
| [颜色](/editor/tools/colors) | 配置纹理处理 |
| [平滑](/editor/tools/smooth) | VRM角色平滑处理 |
| [目标面数](/editor/tools/target-faces) | 控制多边形数量 |

## 交互工具（中间）

这些工具**直接修改您的模型**。在修复后使用它们进行额外更改。

| 工具 | 功能 |
|------|------|
| [切割](/editor/tools/cut) | 使用可选卡扣切割为多块 |
| [晶格](/editor/tools/lattice) | 生成填充结构 |
| [标签](/editor/tools/label) | 在表面添加文字 |
| [镜像](/editor/tools/mirror) | 创建镜像副本 |
| [测量](/editor/tools/measure) | 测量距离 |
| [移除小零件](/editor/tools/remove-small-parts) | 清理浮动碎片 |

## 检查工具（下方）

这些工具帮助您**分析和可视化**模型，不会改变它。

| 工具 | 功能 |
|------|------|
| [验证](/editor/tools/validate) | 可打印性分析 |
| [错误](/editor/tools/errors) | 网格质量报告 |
| [对象信息](/editor/tools/object-info) | 尺寸和统计信息 |
| [360°视图](/editor/tools/360-view) | 灯光和查看选项 |
| [切片视图](/editor/tools/slice-view) | 横截面可视化 |

## 视觉指示

- **蓝色高亮** — 当前活动工具
- **蓝色圆点** — 设置已从默认值自定义的工具
- **灰色** — 需要已修复模型的工具（先修复以解锁）
