平滑在修复模型的同时柔化并圆滑其表面。其中一部分发生在修复**之前**（松弛您的原始模型，使修复从更柔和的形状开始重建），另一部分发生在修复**之后**（仅松弛重建后的表面）。两者不可互换——修复前的平滑会改变输入的形状并影响整个结果，而修复后的平滑会在不触及输入内容的情况下让新表面变得均匀。它们可以组合使用。

```mermaid

    B --> C[修复重建出<br/>干净的表面]:::fix
    C --> M{{间隙平滑}}:::during
    M --> D{{构建后平滑}}:::after
    D --> E([修复后的模型]):::model
```

## 何时使用此工具

平滑适用于任何模型——角色、扫描件和机械部件均可。请根据您想要的表面类型选择设置：

- **角色 / 手办 / 有机模型** — 开启平滑可获得更柔和的表面，减少面部、手部和曲面上的多面化。
- **机械 / CAD 部件** — 保持平滑关闭（或非常低），以使锐利的边缘保持清晰。
- **有噪点的扫描件** — 少量平滑可清理凹凸和扫描伪影。

## 设置

平滑由修复设置中的主**平滑**开关控制。它**默认关闭**，从而使修复后的表面保持最大细节。开启后会显示下面三个控件并应用其推荐值。

### 平滑（主开关）
开启或关闭平滑。**默认值：关闭。**关闭时，三个阶段全部禁用，表面保持其最精细的细节。

### 预平滑 — 修复之前
一个开关，在**原始模型**被修复之前对其进行柔化。这会减少整个最终结果中的多面化和阶梯效应。**开启平滑时自动开启。**由于它会改变进入修复的形状，将其关闭可保留锐利的原始边缘——最适合机械 / CAD 部件；对于有机模型和角色模型请保持开启。

### 间隙平滑（0–256）— 重建期间
一个滑块，**在修复重建实体的同时**对其细节进行圆滑。**默认值：128。**为 0 时保留最精细的细节；较高的值会在新表面最终确定之前去除更多凹凸和噪点。

### 构建后平滑（0–50）— 修复之后
一个滑块，在修复之后松弛**重建后的表面**。这是最常用的平滑控件。**默认值：8。**为 0 时直接从修复中保留精细细节；每提高一步都会让表面更均匀一些，柔化细小特征。

### 快速参考

| 设置 | 作用时机 | 0 / 关闭 | 更高 / 开启 | 开启时的默认值 |
|---|---|---|---|---|
| **预平滑** | 修复之前 | 保留锐利的原始边缘 | 更柔和的表面，更少多面化 | 开启 |
| **间隙平滑** | 重建期间 | 保留最精细的细节 | 圆滑实体上的细节 | 128 |
| **构建后平滑** | 修复之后 | 保留最精细的细节 | 更平滑的表面，细小特征柔化 | 8 |

## 按模型类型选择设置

- **机械 / CAD 部件**（锐利边缘很重要）— 平滑**关闭**，或构建后设为低值（0–2）。保持预平滑**关闭**。
- **角色 / 手办 / 有机模型**（偏好平滑表面）— 平滑**开启**，构建后**8**，预平滑**开启**。
- **有噪点的扫描件** — 用较小的间隙平滑值来清理体积，再加上构建后**8**。

## 提示

- 较高的构建后和间隙值会柔化表面，并可能抹去细小细节，如皱纹、雕刻和薄浮雕——降低它们可保留更多细节。
- 平滑不会改变模型的多边形数量。多边形数量由[目标面数](/editor/tools/target-faces) / 智能多边形数控件单独设置。
- 这些是修复设置——在修复过程中应用，而不是作为独立操作。

<Note>
  VRM / VRoid 文件会根据其文件扩展名自动检测，因此没有单独的 VRM 平滑开关。请像对待任何其他模型一样，对角色模型使用上面的平滑控件。
</Note>
