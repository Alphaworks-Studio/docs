错误检测扫描您的模型，查找可能导致打印失败或视觉伪影的几何问题。

## 何时使用此工具

- 修复前，了解模型有什么问题
- 修复后，验证所有问题是否已解决
- 当模型看起来不对或有意外的缝隙时

## 检测内容

| 错误类型 | 含义 |
|---------|------|
| **非流形边** | 被两个以上表面共享的边——模型不是正确的实体 |
| **自相交** | 相互穿过的表面 |
| **退化面** | 极薄或零面积的三角形 |
| **翻转法线** | 朝内而不是朝外的表面 |

## 结果

- 每种错误类型的**计数**
- 直接在模型上的**高亮位置**——错误在视口中标记
- **点击导航**——点击列表中的错误，相机跳转到该位置

## 提示

- **非流形边**是最常见的问题——意味着表面没有形成正确的实体
- 在**修复前**运行以了解问题的范围
- 修复后，错误计数应为**零或接近零**
- 如果修复后错误仍然存在，尝试不同的[模型类型预设](/repair/model-types)或[质量级别](/repair/quality-levels)
