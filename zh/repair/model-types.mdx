开始修复时，第一步是选择模型类型预设。每个预设都针对特定类型的3D模型优化修复过程。

<CardGroup cols={2}>
  <Card title="角色" icon="user">
    针对手办、角色和有机形状优化。保留面部特征、头发和衣服褶皱等精细细节。使用更温和的平滑处理来保持精致的几何体。
  </Card>
  <Card title="机械" icon="gear">
    最适合工程零件、支架和功能性打印件。优先保留锐利边缘和平坦表面。保持精确尺寸，适用于需要装配的零件。
  </Card>
  <Card title="车辆" icon="car">
    针对汽车、飞机和其他车辆模型调整。平衡车身平滑曲线与车轮和面板等结构细节。
  </Card>
  <Card title="自定义" icon="sliders">
    无预设优化——您可以完全控制每个设置。如果您的模型不适合其他类别，这是一个好的起点。
  </Card>
</CardGroup>

<Tip>
  不确定选哪个？先从**自定义**开始——它适用于大多数模型。您随时可以使用不同的预设重新修复。
</Tip>

<Note>
  如果我们的AI助手已分析过您的模型，它可能会建议推荐的模型类型。您会在修复对话框中看到**推荐**标记。
</Note>
