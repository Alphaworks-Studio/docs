| Format | Extensions | Textures | Colors | Notes |
|--------|-----------|----------|--------|-------|
| Wavefront OBJ | `.obj` + `.mtl` | Yes | Yes | Include .mtl and texture files in a ZIP |
| Stereolithography | `.stl` | No | No | Geometry only, most common for 3D printing |
| 3D Manufacturing | `.3mf` | Yes | Yes | Modern format with embedded materials |
| Autodesk FBX | `.fbx` | Yes | Yes | Common in game/animation workflows |
| Stanford PLY | `.ply` | No | Yes | Supports vertex colors |
| glTF / GLB | `.gltf`, `.glb` | Yes | Yes | Web standard, GLB embeds textures |
| VRM (VRoid) | `.vrm` | Yes | Yes | Anime/VTuber character format |
| VRML | `.wrl` | Yes | Yes | Legacy 3D web format |
| DICOM | `.dcm` | No | Yes | Medical imaging (CT, MRI) |
| LAS / LAZ | `.las`, `.laz` | No | Yes | LiDAR point clouds |
