Espelho cria uma cópia invertida do seu modelo ao longo de um eixo escolhido e opcionalmente a mescla com o original. Útil para criar objetos simétricos ou duplicar peças.

## Quando usar esta ferramenta

- Criar uma versão esquerda a partir de uma peça direita
- Construir objetos simétricos a partir de uma metade
- Duplicar um modelo na direção oposta

## Configurações

### Eixo
**X**, **Y** ou **Z** — o eixo ao longo do qual espelhar.

### Espaçamento (mm)
O espaço entre o original e a cópia espelhada. Quando o **Deslocamento automático** está ligado, este é o espaçamento usado para evitar que as duas metades se sobreponham. Padrão: 2.

### Mesclar
Alternar (padrão: ligado). Combina o original e a cópia espelhada em um único objeto imprimível.

### Inverter UVs
Alternar (padrão: desligado). Também espelha as coordenadas de textura. Ative se as texturas do lado espelhado parecerem erradas.

## Passo a passo

<Steps>
  <Step title="Selecione a ferramenta Espelho">
    Clique em **Espelho** na seção Ferramentas Interativas.
  </Step>
  <Step title="Escolha o eixo">
    Selecione X, Y ou Z.
  </Step>
  <Step title="Ajuste o espaçamento se necessário">
    Defina o **Espaçamento** para controlar o espaço entre o original e a cópia espelhada.
  </Step>
  <Step title="Aplicar">
    Clique em **Aplicar** para criar o espelho.
  </Step>
</Steps>

## Dicas

- **Mesclar** cria um único objeto imprimível; sem mesclar você obtém duas peças separadas
- Ative **Inverter UVs** se texto ou padrões assimétricos aparecerem invertidos no lado espelhado
