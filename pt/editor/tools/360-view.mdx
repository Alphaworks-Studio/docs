Vista 360° mostra as miniaturas pré-renderizadas do seu modelo de múltiplos ângulos e permite personalizar a iluminação e o fundo do viewport para melhor visualização.

## Configurações

### Preset de iluminação
- **Estúdio** — Iluminação balanceada e profissional
- **Natural** — Iluminação suave, estilo exterior
- **Dramática** — Alto contraste com sombras fortes
- **Plana** — Iluminação uniforme, mostra cores com maior precisão

### Tema de fundo
- **Escuro** — Fundo escuro (padrão)
- **Claro** — Fundo branco/claro
- **Degradê** — Fundo com degradê suave
- **Transparente** — Sem fundo

### Rotação automática
Alterne a rotação automática com velocidade ajustável. Ótimo para ter uma visão completa do seu modelo sem movimento manual do mouse.

## Dicas

- Use iluminação **Dramática** para destacar detalhes de superfície e potenciais problemas de impressão
- Use iluminação **Plana** para ver cores com maior precisão — útil para impressões coloridas
- Ative a **rotação automática** para uma visão geral rápida de todo o modelo
