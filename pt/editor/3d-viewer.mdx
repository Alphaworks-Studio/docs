## Controles de câmera

| Ação | Mouse | Trackpad |
|------|-------|----------|
| **Girar** (órbita) | Clique esquerdo + arrastar | Arrastar com dois dedos |
| **Deslocar** | Clique direito + arrastar | Shift + arrastar com dois dedos |
| **Ampliar** | Roda de rolagem | Pinçar |

<Tip>
  Clique duplo em qualquer parte do modelo para centralizar a câmera nessa área.
</Tip>

## ViewCube

</Frame>

O **ViewCube** é um pequeno cubo no canto do viewport. Ele mostra seu ângulo de visualização atual e permite mudar rapidamente para vistas padrão:
- Clique em uma **face** (Frontal, Traseira, Esquerda, Direita, Superior, Inferior) para ir a essa vista
- Clique em uma **aresta** para uma vista diagonal
- Clique em um **canto** para uma perspectiva isométrica

## Modos de exibição

Alterne entre diferentes estilos de renderização:
- **Com textura** — Cores e texturas completas (padrão)
- **Sólido** — Cor de material único, bom para inspecionar a forma
- **Wireframe** — Mostra as arestas dos polígonos, útil para verificar a densidade da geometria

## Comparação lado a lado

Após um reparo, o editor pode mostrar os modelos **original** e **reparado** lado a lado. Ambas as vistas giram e ampliam juntas, facilitando identificar exatamente o que mudou.

## Seleção interativa

Algumas ferramentas permitem clicar diretamente no modelo:
- **Ferramenta de medição** — Clique para posicionar pontos de medição
- **Ferramenta de etiqueta** — Clique para definir a posição da etiqueta
- **Ferramenta de corte** — Arraste o manipulador do plano de corte para posicioná-lo

Quando uma ferramenta de seleção está ativa, o cursor muda para uma mira.
