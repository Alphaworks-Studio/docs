A barra de ferramentas no lado direito do editor organiza 19 ferramentas em três grupos, separados por divisores.

## Ferramentas de processamento (superior)

Essas configurações afetam o **processo de reparo**. Ajuste-as antes de executar um reparo.

| Ferramenta | Função |
|------------|--------|
| [Escala](/editor/tools/scale) | Redimensionar seu modelo |
| [Espessura de parede](/editor/tools/thickness) | Definir espessura mínima de parede |
| [Esvaziamento](/editor/tools/hollow) | Criar interior oco |
| [Fechar buracos](/editor/tools/close-holes) | Preencher lacunas e aberturas |
| [Cortar transparente](/editor/tools/cut-transparent) | Remover superfícies invisíveis |
| [Cores](/editor/tools/colors) | Configurar tratamento de texturas |
| [Suavização](/editor/tools/smooth) | Suavização de personagens VRM |
| [Faces alvo](/editor/tools/target-faces) | Controlar quantidade de polígonos |

## Ferramentas interativas (centro)

Essas ferramentas **modificam seu modelo diretamente**. Use-as após o reparo para alterações adicionais.

| Ferramenta | Função |
|------------|--------|
| [Cortar](/editor/tools/cut) | Dividir em peças com encaixes opcionais |
| [Preenchimento reticular](/editor/tools/lattice) | Gerar estruturas de preenchimento |
| [Etiqueta](/editor/tools/label) | Adicionar texto à superfície |
| [Espelho](/editor/tools/mirror) | Criar uma cópia espelhada |
| [Medição](/editor/tools/measure) | Medir distâncias |
| [Remover peças pequenas](/editor/tools/remove-small-parts) | Limpar fragmentos soltos |

## Ferramentas de inspeção (inferior)

Essas ferramentas ajudam a **analisar e visualizar** seu modelo sem modificá-lo.

| Ferramenta | Função |
|------------|--------|
| [Validar](/editor/tools/validate) | Análise de imprimibilidade |
| [Erros](/editor/tools/errors) | Relatório de qualidade da malha |
| [Informações do objeto](/editor/tools/object-info) | Dimensões e estatísticas |
| [Vista 360°](/editor/tools/360-view) | Opções de iluminação e visualização |
| [Vista de corte](/editor/tools/slice-view) | Visualização de seções transversais |

## Indicadores visuais

- **Destaque azul** — A ferramenta atualmente ativa
- **Ponto azul** — Uma ferramenta cujas configurações foram personalizadas
- **Esmaecida** — Uma ferramenta que requer um modelo reparado (repare primeiro para desbloquear)
