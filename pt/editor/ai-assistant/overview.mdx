O Assistente IA é um assistente de chat integrado que pode ver seu modelo, entender suas propriedades e ajudá-lo a prepará-lo para impressão. Ele pode responder perguntas, sugerir correções e até executar ferramentas para você.

## Abrindo o Assistente IA

Clique no **ícone de brilho** no canto inferior direito da barra de ferramentas do editor. O painel de chat desliza para abrir no lado direito da tela.

## O que ele pode ver

Quando você inicia uma conversa, o Assistente IA automaticamente tem acesso a:
- As dimensões, contagem de polígonos e volume do seu modelo
- Resultados de análise (se disponíveis) — problemas detectados, pontuação de imprimibilidade
- Imagens de miniatura 360° do seu modelo de múltiplos ângulos
- O estado atual das configurações do seu editor

Isso significa que você pode perguntar sobre seu modelo diretamente — sem necessidade de descrevê-lo.

## Como usar

Digite uma mensagem na caixa de entrada na parte inferior e pressione **Enter** ou clique em **Enviar**. O assistente responde em tempo real, e você pode ter uma conversa com múltiplas rodadas.

<Frame>
  <img src="/images/editor/ai/quick-prompts.png" alt="Sugestões de prompts rápidos no Assistente IA" />

**Prompts rápidos** — Quando você abre o chat pela primeira vez, verá sugestões de prompts para começar:
- "Analisar este modelo para imprimibilidade"
- "O que devo fazer para preparar isto para impressão?"
- "Há algum problema com este modelo?"

Clique em qualquer um destes para enviá-lo imediatamente.

## Chamadas de ferramentas

O Assistente IA pode executar ferramentas no seu modelo como parte da conversa. Quando o fizer, você verá um **cartão de chamada de ferramenta** no chat mostrando:
- Qual ferramenta está sendo usada
- Os parâmetros que ele escolheu
- O resultado quando a operação é concluída

Você pode expandir esses cartões para ver os detalhes.

## Gerenciando conversas

- **Parar** — Clique no botão vermelho de parar para interromper uma resposta
- **Limpar** — Clique no botão de nova conversa no cabeçalho para começar do zero
- Conversas são salvas automaticamente para que você possa continuar de onde parou

<Tip>
  O Assistente IA é ótimo para obter uma avaliação rápida de imprimibilidade. Basta perguntar "Este modelo está pronto para imprimir?" e ele analisará e dará recomendações específicas.
</Tip>
