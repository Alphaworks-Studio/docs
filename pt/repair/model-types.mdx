Quando você inicia um reparo, o primeiro passo é escolher um preset de tipo de modelo. Cada preset otimiza o processo de reparo para um tipo específico de modelo 3D.

<CardGroup cols={2}>
  <Card title="Personagem" icon="user">
    Otimizado para miniaturas, personagens e formas orgânicas. Preserva detalhes finos como características faciais, cabelo e dobras de roupa. Usa suavização mais suave para manter geometria delicada.
  </Card>
  <Card title="Mecânico" icon="gear">
    Melhor para peças de engenharia, suportes e impressões funcionais. Prioriza bordas afiadas e superfícies planas. Mantém dimensões precisas para peças que precisam encaixar juntas.
  </Card>
  <Card title="Veículo" icon="car">
    Ajustado para carros, aeronaves e outros modelos de veículos. Equilibra curvas suaves do corpo com detalhes estruturais como rodas e painéis.
  </Card>
  <Card title="Personalizado" icon="sliders">
    Sem otimizações predefinidas — você tem controle total sobre cada configuração. Bom ponto de partida se seu modelo não se encaixa nas outras categorias.
  </Card>
</CardGroup>

<Tip>
  Não tem certeza de qual escolher? Comece com **Personalizado** — funciona bem para a maioria dos modelos. Você sempre pode re-reparar com um preset diferente depois.
</Tip>

<Note>
  Se nosso assistente IA analisou seu modelo, ele pode sugerir um tipo de modelo recomendado. Você verá um selo **Recomendado** no diálogo de reparo.
</Note>
