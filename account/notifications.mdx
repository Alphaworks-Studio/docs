Control your email notification preferences from Account Settings. Toggle each notification type on or off.

## Notification types

| Notification | Description | Default |
|---|---|---|
| **Model Repaired** | Get notified when a repair finishes processing | On |
| **Product Updates** | News about new features and improvements | On |
| **Promotional Offers** | Special discounts and promotions | Off |

Changes save automatically when you toggle a notification.

<Tip>
  Keep "Model Repaired" enabled if you run repairs on large models that take several minutes. You'll get an email as soon as the result is ready.
</Tip>
