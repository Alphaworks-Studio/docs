Ziel-Polygonanzahl ermöglicht es Ihnen, eine benutzerdefinierte Polygonanzahl für die reparierte Ausgabe festzulegen. Die Reduzierung von Polygonen erzeugt kleinere Dateien, die schneller gesliced werden, während das Beibehalten der Originalanzahl maximale Details bewahrt.

## Wann Sie dies verwenden

- Wenn Ihr Modell Millionen von Polygonen hat und Sie eine leichtere Datei möchten
- Wenn Ihr Slicer mit hochpolygonigen Modellen Schwierigkeiten hat
- Wenn Sie eine bestimmte Polygonanzahl für Ihren Workflow benötigen

## Einstellungen

### Originalanzahl verwenden
Schalter. **Standard: Ein** — behält die originale Polygonanzahl bei.

### Zielanzahl
Bei Deaktivierung geben Sie eine benutzerdefinierte Zahl ein (3–5.000.000).

## Tipps

- **100.000–500.000** Flächen sind für die meisten Drucke ausreichend detailliert
- Unter **50.000** können sichtbare feine Merkmale wie Gesichtsdetails verloren gehen
- Höhere Polygonanzahlen erzeugen glattere Oberflächen, aber größere Dateien
- Prüfen Sie die aktuelle Anzahl mit dem [Objektinfo](/editor/tools/object-info)-Werkzeug
