## Fortschritt verfolgen

Nachdem Sie auf **Reparatur starten** geklickt haben, erscheint ein Vollbild-Fortschritts-Overlay.

<Frame>
  <img src="/images/getting-started/repair-progress.png" alt="Fortschritts-Overlay der Reparatur mit aktuellem Stadium und Fortschrittsbalken" />
</Frame>

Sie sehen:
- Einen **Fortschrittsbalken** von 0% bis 100%
- Den **aktuellen Phasennamen** (z.B. „Geometrie analysieren", „Korrekturen anwenden")
- Eine **Beschreibung** dessen, was gerade passiert

## Reparatur abbrechen

Klicken Sie jederzeit auf die **Abbrechen**-Schaltfläche, um die Reparatur zu stoppen. Ihr Originalmodell wird niemals verändert — Sie können jederzeit mit anderen Einstellungen von vorne beginnen.

## Wenn die Reparatur abgeschlossen ist

Sie werden automatisch zur **Reparatur-Detailseite** weitergeleitet, wo Sie:
- Den [KI-Reparaturbericht](/repair/ai-fix-report) einsehen können — was gefunden und was behoben wurde
- Original- und repariertes Modell [nebeneinander vergleichen](/editor/3d-viewer) können
- Das Ergebnis in Ihrem bevorzugten Format [herunterladen](/getting-started/download-results) können
- Das reparierte Modell im [3D-Editor](/editor/overview) für weitere Bearbeitungen öffnen können

<Frame>
  <img src="/images/getting-started/repair-complete.png" alt="Abgeschlossene Reparatur-Detailseite mit Reparaturbericht" />
</Frame>

## Wenn eine Reparatur fehlschlägt

Gelegentlich kann eine Reparatur nicht erfolgreich abgeschlossen werden. In diesem Fall:
- Versuchen Sie eine niedrigere [Qualitätsstufe](/repair/quality-levels) (Schnell statt Detailliert)
- Versuchen Sie eine andere [Modelltypvorlage](/repair/model-types)
- Prüfen Sie, ob Ihre Modelldatei beschädigt ist, indem Sie sie in einem anderen 3D-Viewer öffnen
- Weitere Lösungen finden Sie unter [Fehlerbehebung](/help/troubleshooting)

<Note>
  Ihr Originalmodell bleibt immer erhalten. Reparaturen erstellen eine neue Kopie — sie verändern niemals Ihren Upload.
</Note>
