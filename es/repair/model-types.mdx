Cuando inicia una reparación, el primer paso es elegir una plantilla de tipo de modelo. Cada plantilla optimiza el proceso de reparación para un tipo específico de modelo 3D.

<CardGroup cols={2}>
  <Card title="Personaje" icon="user">
    Optimizado para figuras, personajes y formas orgánicas. Preserva detalles finos como rasgos faciales, cabello y pliegues de ropa. Usa un grosor de pared mínimo más delgado (0.7 mm) para mantener la geometría delicada, un espacio de holgura de 1.5 mm para la fusión, recorte de texturas transparentes y mezcla de textura/material. No aplica suavizado — el suavizado es una opción separada que se activa manualmente.
  </Card>
  <Card title="Mecánico" icon="gear">
    Ideal para piezas de ingeniería, soportes e impresiones funcionales. Prioriza bordes afilados y superficies planas. Mantiene dimensiones precisas para piezas que necesitan encajar entre sí.
  </Card>
  <Card title="Vehículo" icon="car">
    Ajustado para coches, aviones y otros modelos de vehículos. Equilibra curvas suaves de carrocería con detalles estructurales como ruedas y paneles.
  </Card>
  <Card title="Personalizado" icon="sliders">
    Sin optimizaciones predefinidas — tiene control total sobre cada configuración. Buen punto de partida si su modelo no encaja en las otras categorías.
  </Card>
</CardGroup>

<Tip>
  ¿No está seguro cuál elegir? Comience con **Personalizado** — funciona bien para la mayoría de los modelos. Siempre puede volver a reparar con una plantilla diferente después.
</Tip>

<Note>
  Una vez que su modelo ha sido analizado, una tarjeta **Recomendado** reemplaza la tarjeta **Personalizado** en la lista de tipos de modelo. Está preseleccionada cuando se abre el diálogo de reparación y rellena previamente cada configuración de reparación — incluida la activación del suavizado — a partir del análisis. Internamente, una reparación recomendada se ejecuta como una reparación personalizada.
</Note>
