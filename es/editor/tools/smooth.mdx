Suavizado activa un proceso especial diseñado para modelos de personajes VRM (VRoid). Aplica subdivisión para reducir la apariencia angular y de pocos polígonos común en personajes 3D de estilo anime.

## Cuándo usar esta herramienta

Cuando imprime personajes VRM o VRoid que se ven angulares o facetados — especialmente caras, manos y superficies curvas que aparecen como formas poligonales planas.

## Ajustes

### Activar
Activar o desactivar. **Predeterminado: Desactivado.**

## Consejos

- Solo útil para modelos en formato VRM/VRoid — tiene efecto mínimo en modelos que ya son suaves
- Aumenta significativamente el conteo de polígonos, lo que puede incrementar el tiempo de procesamiento
- Este es un ajuste de procesamiento — se aplica durante la reparación, no como una operación independiente

<Note>
  Si no está imprimiendo un personaje VRM, puede ignorar esta herramienta.
</Note>
