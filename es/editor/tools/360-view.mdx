Vista 360° muestra las miniaturas pre-renderizadas de su modelo desde múltiples ángulos y le permite personalizar la iluminación y el fondo de la ventana gráfica para una mejor visualización.

## Ajustes

### Preset de iluminación
- **Estudio** — Iluminación equilibrada y profesional
- **Natural** — Iluminación suave, estilo exterior
- **Dramática** — Alto contraste con sombras fuertes
- **Plana** — Iluminación uniforme, muestra los colores con más precisión

### Tema de fondo
- **Oscuro** — Fondo oscuro (predeterminado)
- **Claro** — Fondo blanco/claro
- **Gradiente** — Fondo degradado suave
- **Transparente** — Sin fondo

### Auto-rotación
Active la rotación automática con velocidad ajustable. Ideal para obtener una vista completa de su modelo sin movimiento manual del ratón.

## Consejos

- Use iluminación **Dramática** para resaltar detalles de superficie y posibles problemas de impresión
- Use iluminación **Plana** para ver los colores con más precisión — útil para impresiones a color completo
- Active la **auto-rotación** para una vista rápida general de todo el modelo
