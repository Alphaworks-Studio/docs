Espejo crea una copia invertida de su modelo a lo largo de un eje elegido y opcionalmente la fusiona con el original. Útil para crear objetos simétricos o duplicar piezas.

## Cuándo usar esta herramienta

- Crear una versión izquierda a partir de una pieza derecha
- Construir objetos simétricos a partir de una mitad
- Duplicar un modelo en la dirección opuesta

## Ajustes

### Eje
**X**, **Y** o **Z** — el eje de reflexión.

### Espaciado (mm)
El espacio entre el original y la copia reflejada. Cuando **Desplazamiento automático** está activado, este es el espaciado utilizado para evitar que las dos mitades se superpongan. Predeterminado: 2.

### Fusionar
Interruptor (predeterminado: activado). Combina el original y la copia reflejada en un solo objeto imprimible.

### Invertir UVs
Interruptor (predeterminado: desactivado). También refleja las coordenadas de textura. Active si las texturas del lado reflejado se ven incorrectas.

## Paso a paso

<Steps>
  <Step title="Seleccione la herramienta Espejo">
    Haga clic en **Espejo** en la sección de Herramientas interactivas.
  </Step>
  <Step title="Elija el eje">
    Seleccione X, Y o Z.
  </Step>
  <Step title="Ajuste el espaciado si es necesario">
    Establezca el **Espaciado** para controlar el espacio entre el original y la copia reflejada.
  </Step>
  <Step title="Aplicar">
    Haga clic en **Aplicar** para crear el espejo.
  </Step>
</Steps>

## Consejos

- **Fusionar** crea un solo objeto imprimible; sin fusionar obtiene dos piezas separadas
- Active **Invertir UVs** si texto o patrones asimétricos aparecen al revés en la mitad reflejada
