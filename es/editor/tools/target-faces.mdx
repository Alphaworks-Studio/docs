Caras objetivo le permite establecer una cantidad personalizada de polígonos para el resultado reparado. Reducir polígonos genera archivos más pequeños que se procesan más rápido en el software de corte, mientras que mantener la cantidad original preserva el máximo detalle.

## Cuándo usar esta herramienta

- Cuando su modelo tiene millones de polígonos y desea un archivo más ligero
- Cuando su software de corte tiene dificultades con modelos de muchos polígonos
- Cuando necesita una cantidad específica de polígonos para su flujo de trabajo

## Ajustes

### Usar caras originales
Interruptor. **Predeterminado: Activado** — mantiene la cantidad original de polígonos.

### Cantidad objetivo
Cuando está desactivado, ingrese un número personalizado (3–5,000,000).

## Consejos

- **100,000–500,000** caras es suficiente detalle para la mayoría de las impresiones
- Reducir por debajo de **50,000** puede perder detalles finos visibles como rasgos faciales
- Más polígonos producen superficies más suaves pero archivos más grandes
- Verifique la cantidad actual con la herramienta [Información del objeto](/editor/tools/object-info)
