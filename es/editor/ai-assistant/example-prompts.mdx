Aquí tiene ideas de prompts organizadas por lo que desea lograr. Copie cualquiera de estos o úselos como inspiración para sus propias preguntas.

## Analizar su modelo

- "¿Está este modelo listo para imprimir?"
- "¿Qué problemas tiene este modelo?"
- "Analizar esto para imprimibilidad en una impresora SLA"
- "¿Hay paredes delgadas que podrían romperse?"
- "¿Cuántas partes separadas tiene este modelo?"

## Reparación

- "Reparar este modelo con calidad estándar"
- "Arreglar esto para mi impresora Mimaki a 8 pulgadas de alto"
- "Reparar con 0.7mm de grosor de pared y 2mm de ahuecado"
- "Reparar este modelo de personaje a 150mm de altura"
- "Arreglar esto y mantener la cantidad original de polígonos"

## Cortar y dividir

- "Cortar esto a la mitad a lo largo del eje Y"
- "Dividir esto en dos piezas con ensamblajes de pasador"
- "Cortar horizontalmente al 60% de altura con ensamblajes de cola de milano"
- "Dividir este modelo en mitad superior e inferior"

## Reticulado y relleno

- "Agregar un reticulado giroide con celdas de 5mm"
- "Llenar el interior con un reticulado cúbico"
- "Crear un patrón de relleno diamante con celdas de 8mm"
- "Agregar relleno reticular con barras de 0.8mm de grosor"

## Etiquetas y marcas

- "Grabar 'Muestra' en la base"
- "Agregar mi nombre a la base — extruir 0.5mm"
- "Poner el texto 'v3' en la parte trasera de este modelo"

## Medición

- "¿Cuáles son las dimensiones generales?"
- "¿Qué altura tiene este modelo?"
- "Medir la distancia entre los dos agujeros"
- "¿Cuál es el volumen de este modelo?"

## Orientación general

- "¿Qué debo hacer para preparar esto para la impresión?"
- "Sugerir mejoras para este modelo"
- "¿Cuál es la mejor manera de imprimir este modelo?"
- "Este modelo se ve áspero — ¿qué puedo hacer?"
- "¿Debería ahuecatr este modelo?"

<Tip>
  Sea específico cuando pueda. "Reparar esto a 6 pulgadas de alto con paredes de 1mm" obtiene mejores resultados que solo "Arreglar esto." El IA también puede trabajar con solicitudes vagas, pero las específicas ahorran tiempo.
</Tip>

<Note>
  El Asistente IA recuerda el historial de conversación dentro de una sesión. Puede decir cosas como "Ahora cortar el modelo reparado a la mitad" y sabe a qué se refiere.
</Note>
