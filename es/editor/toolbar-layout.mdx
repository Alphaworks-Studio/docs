La barra de herramientas en el lado derecho del editor organiza 19 herramientas en tres grupos, separados por divisores.

## Herramientas de procesamiento (superior)

Estos ajustes afectan al **proceso de reparación**. Configúrelos antes de ejecutar una reparación.

| Herramienta | Función |
|-------------|---------|
| [Escala](/editor/tools/scale) | Redimensionar su modelo |
| [Grosor de pared](/editor/tools/thickness) | Establecer el grosor mínimo de pared |
| [Ahuecado](/editor/tools/hollow) | Crear un interior hueco |
| [Cerrar agujeros](/editor/tools/close-holes) | Rellenar huecos y aberturas |
| [Cortar transparente](/editor/tools/cut-transparent) | Eliminar superficies invisibles |
| [Colores](/editor/tools/colors) | Configurar el manejo de texturas |
| [Suavizado](/editor/tools/smooth) | Suavizado de personajes VRM |
| [Caras objetivo](/editor/tools/target-faces) | Controlar la cantidad de polígonos |

## Herramientas interactivas (centro)

Estas herramientas **modifican su modelo directamente**. Utilícelas después de la reparación para realizar cambios adicionales.

| Herramienta | Función |
|-------------|---------|
| [Cortar](/editor/tools/cut) | Dividir en piezas con ensamblajes opcionales |
| [Relleno reticular](/editor/tools/lattice) | Generar estructuras de relleno |
| [Etiqueta](/editor/tools/label) | Agregar texto a la superficie |
| [Espejo](/editor/tools/mirror) | Crear una copia reflejada |
| [Medición](/editor/tools/measure) | Medir distancias |
| [Eliminar piezas pequeñas](/editor/tools/remove-small-parts) | Limpiar fragmentos sueltos |

## Herramientas de inspección (inferior)

Estas herramientas le ayudan a **analizar y visualizar** su modelo sin modificarlo.

| Herramienta | Función |
|-------------|---------|
| [Validar](/editor/tools/validate) | Análisis de imprimibilidad |
| [Errores](/editor/tools/errors) | Informe de calidad de la malla |
| [Información del objeto](/editor/tools/object-info) | Dimensiones y estadísticas |
| [Vista 360°](/editor/tools/360-view) | Opciones de iluminación y visualización |
| [Vista de corte](/editor/tools/slice-view) | Visualización de secciones transversales |

## Indicadores visuales

- **Resaltado azul** — La herramienta actualmente activa
- **Punto azul** — Una herramienta cuya configuración ha sido personalizada
- **Atenuada** — Una herramienta que requiere un modelo reparado (repare primero para desbloquearla)
